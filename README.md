# Web Portfolio

A personal portfolio website for Delcine Mawoyo, an Information Security and Assurance student specializing in web design and cyber security.

## Description

This multi-page portfolio showcases skills, projects, and experience in web development and information security. Built with semantic HTML5 and CSS3 for a responsive, mobile-first design.

## Pages

- **Home (index.html)**: Hero section with introduction, stats, and call-to-action buttons.
- **About (about.html)**: Biography, skills grid, and educational background.
- **Projects (projects.html)**: Showcase of selected projects with links to GitHub and live demos.
- **Contact (contact.html)**: Contact form, email, location, and social media links.

## Technologies Used

- HTML5
- CSS3
- Responsive Design (Mobile-first)

## Installation & Usage

1. Clone the repository:
   ```
   git clone https://github.com/Delcy22/webportfolio.git
   ```

2. Open `index.html` in your web browser to view the portfolio.

No build process required — it's a static site.
